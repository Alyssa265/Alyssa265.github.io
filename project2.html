Description

The CatBrew Cafe is a simulation game created in Scratch using Codeblocking. It includes interactive features that allow users to follow a recipe to serve cat-themed coffee and boba drinks. The game has a time limit feature to increase difficulty and challenge players’ multitasking skills.

Skills Learned

  • Codeblocking and programming logic in Scratch
  • Interactive game design principles
  • User interface (UI) and user experience (UX) considerations
  • Time management and task sequencing in games
  • Teamwork and collaboration during project development
